Это мое приложение WPF. 

Оно реализует базовый функционал приложения для рисования(собственно возможность рисовать, выбирать цвет кисти, выделять обьекты а также два варината ластика),
возможность просматривать альбомы с картинками и видео, переключаться между отдельными фотографиями и видео а также возможнсоть удалять их,
и наконец возможность создать или открыть обычный текстовый файл.
